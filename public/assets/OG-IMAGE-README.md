# Social Sharing Image (OG Image)

This folder should contain an `og-image.png` file for social media sharing.

## Recommended Specifications:
- **Size**: 1200 x 630 pixels
- **Format**: PNG or JPG
- **File size**: Under 1MB for optimal loading

## Design Suggestions:
- Include the Tunnels.ng logo
- Use the brand colors (Orange #F97316, Dark backgrounds)
- Add tagline: "Technology Venture Studio - Build Now, Pay Later"
- Keep text minimal and readable

## Usage:
This image appears when the site is shared on:
- Facebook
- Twitter/X
- LinkedIn
- WhatsApp
- Other social platforms

Upload your `og-image.png` to `/public/assets/` folder.
